# About the Job

Senior Software Engineer -- Platform Team

Full-time

Location: Remote (US)

Reports To: Engineering Manager, Platform

## Why This Role Matters

Example Corp is building the next generation of our e-commerce platform, serving 50M+ monthly active users. We need a Senior Software Engineer to help scale our backend services, improve developer experience, and drive architectural decisions across the platform team.

## Technology Stack

- Go, Python
- AWS (ECS, Lambda, DynamoDB, S3, CloudFormation)
- Kubernetes, Docker, Terraform
- PostgreSQL, Redis, Kafka
- GitHub Actions, Datadog

## Key Responsibilities

### Backend Development
- Design and implement scalable microservices in Go and Python
- Build and maintain APIs serving millions of requests per day
- Optimize database queries and caching strategies for performance

### Platform & Infrastructure
- Improve CI/CD pipelines and developer tooling
- Manage Kubernetes clusters and containerized workloads
- Implement infrastructure-as-code using Terraform

### Technical Leadership
- Mentor junior and mid-level engineers through code reviews and pairing
- Lead architectural discussions and write technical design documents
- Participate in on-call rotation and incident response

## Required Experience

- 5+ years of backend software engineering experience
- Strong proficiency in Go or Python
- Experience with AWS cloud services
- Experience with containerization (Docker, Kubernetes)
- Track record of building and scaling distributed systems

## Preferred Experience

- Experience with event-driven architectures (Kafka, SQS, SNS)
- Infrastructure-as-code experience (Terraform, CloudFormation)
- Open-source contributions
- Experience mentoring other engineers
- Certifications: AWS Solutions Architect, CKA

## What We Offer

- Competitive salary ($180K-$220K) + equity
- Fully remote with quarterly team offsites
- $3,000 annual learning budget
- Health, dental, vision insurance
